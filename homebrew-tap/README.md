# Homebrew Tap for WUT

This is the official Homebrew tap for [WUT](https://github.com/thirawat27/wut) - AI-Powered Command Helper.

## Installation

```bash
# Add the tap
brew tap thirawat27/wut

# Install WUT
brew install wut
```

## Upgrade

```bash
brew upgrade wut
```

## Uninstall

```bash
brew uninstall wut
brew untap thirawat27/wut
```

## Formula

The formula is automatically generated and updated by [GoReleaser](https://goreleaser.com/) on each release.
