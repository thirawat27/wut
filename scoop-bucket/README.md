# Scoop Bucket for WUT

This is the official Scoop bucket for [WUT](https://github.com/thirawat27/wut) - AI-Powered Command Helper.

## Installation

```powershell
# Add the bucket
scoop bucket add wut https://github.com/thirawat27/scoop-wut

# Install WUT
scoop install wut
```

## Upgrade

```powershell
scoop update wut
```

## Uninstall

```powershell
scoop uninstall wut
scoop bucket rm wut
```

## Manifest

The manifest is automatically generated and updated by [GoReleaser](https://goreleaser.com/) on each release.
